# Create

This folder contains templates and standards for creating assets in the Nice ecosystem.

Each file documents the canonical patterns for a specific asset type. Use these as reference when creating new instances of that type.

---

## Files

| File | When to Read |
|------|--------------|
| `component.md` | Creating a new `nice-react-*` component package |
| `configuration.md` | Setting up rollup, tsconfig, or jest in a package |
| `storybook.md` | Adding stories to `nice-storybook` |

---

## Usage

1. Identify the asset type you need to create
2. Read the corresponding file
3. Follow the patterns exactly
4. If you encounter a case not covered, document it before proceeding
